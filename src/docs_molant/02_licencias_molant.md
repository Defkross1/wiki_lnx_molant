# 02. Licenciamiento y Software Libre

## Conceptos de Licenciamiento
- **Software Libre / Copyleft (GPL):** Garantiza las libertades de uso, estudio, distribución y modificación, con la condición de que las versiones derivadas mantengan la misma licencia libre.
- **Permisivas (MIT, Apache, BSD):** Ofrecen máxima libertad al usuario; se permite modificar e integrar el código en software privativo sin obligar a liberar los cambios.
- **Licencia Propietaria:** El código fuente es cerrado, y su uso está limitado por derechos de autor y contratos comerciales de pago.

## Ubuntu Server y Nginx
- **Ubuntu Server:** Se distribuye principalmente bajo la Licencia Pública General de GNU (GPL), garantizando transparencia y seguridad.
- **Nginx:** Distribuido bajo una licencia simplificada de tipo BSD de 2 cláusulas, altamente permisiva.
