## Objetivo del Laboratorio
El presente laboratorio tiene como finalidad el despliegue, la administración y la fortificación (hardening) de un entorno de servidor corporativo basado en Linux Server (Ubuntu Server 24.04 LTS), gestionado en su totalidad mediante interfaz de línea de comandos (CLI) remota sobre SSH.

## Topología de Red
- **Máquina Anfitriona:** PC con Windows.
- **Servidor:** VM srv-wiki (Ubuntu Server).
- **Adaptador de Red:** NAT con Reenvío de Puertos (TCP 2222 -> 22 y TCP 8080 -> 80).
