# Wiki Molant

Este proyecto corresponde a la creación de una pequeña wiki utilizando archivos en formato Markdown. La idea principal es organizar la información de forma clara y ordenada, facilitando su lectura y navegación.

---

## Objetivo

El objetivo de este trabajo es aprender a:

- Organizar información en archivos `.md`
- Utilizar correctamente rutas e imágenes
- Mantener una estructura ordenada de carpetas
- Documentar procesos realizados durante el laboratorio

---

## Estructura del proyecto

El proyecto se encuentra dividido en las siguientes carpetas:


docs_molant/
img_molant/
src/


- **docs_molant/**: contiene el desarrollo del trabajo en formato Markdown  
- **img_molant/**: almacena las capturas utilizadas como evidencia  
- **src/**: incluye los archivos principales de la wiki  

---

## Uso

Para revisar el contenido, basta con abrir los archivos `.md` ubicados en la carpeta `src`.  
Estos contienen la información organizada por secciones.

También se pueden visualizar directamente en GitHub, donde se respeta el formato Markdown.

---

## Consideraciones

- Las imágenes deben estar correctamente vinculadas desde la carpeta correspondiente  
- Se recomienda mantener nombres claros en archivos y carpetas  
- Es importante que los documentos tengan títulos y subtítulos para facilitar la lectura
